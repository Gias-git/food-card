# Food_Card  

A modern **Restauret** built with MERN. 


## 🌟 Features  

- User authentication (Login/Signup)  
- food add by admin
- User can buy food
- Catagories food view

## 🛠️ Technologies Used  

- **Frontend:** [React, TailwindCSS]  
- **Backend:** [ Node.js, Express]  
- **Database:** [ MongoDB]  
- **Others:** [ Firebase]
